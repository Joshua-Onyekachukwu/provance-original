# Missing Components

Last updated: 2026-07-16

## Purpose

This document lists the major missing components between the current MVP and the longer-term platform vision.

## Product Components

- richer evidence timeline
- share links
- full PDF export workflow
- team and organization collaboration
- enterprise API product
- video support
- audio support

## Engineering Components

- hardened cookie-based session transport
- broader RLS coverage
- fuller observability and alerting
- more mature analytics and telemetry

## Design Components

- stronger premium typographic system
- more cohesive token system
- tighter component consistency
- stronger differentiation in public presentation

## Operational Components

- richer release governance over time
- stronger enterprise compliance documentation
- expanded support knowledge base
