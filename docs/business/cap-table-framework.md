# Cap Table Framework

## Purpose

This document provides a founder-friendly cap table model for Provance.

It is a planning framework, not a legal record.

## Guiding Principles

- preserve founder control early
- avoid stacking too many SAFEs without visibility
- leave room for an employee option pool
- optimize for long-term cap table health, not vanity valuation

## Starting Assumption

Solo founder structure at formation:

- Founder: `100.0%`

## Recommended Early Equity Architecture

### Formation Stage

- Founder common stock: `100.0%`

### Before Institutional Pre-Seed

Recommended reserve planning:

- future employee option pool target: `8% to 12%`

Do not necessarily create the full pool before it is useful, but model it early.

## Suggested Base Case Scenario

## Step 1: Pre-Seed SAFE

Assumption:

- raise: `$650k`
- post-money SAFE cap: `$6M`

Implied ownership at conversion:

- SAFE investors: about `10.8%`

## Step 2: Option Pool Creation

Assumption:

- create / expand option pool to `10%`

## Step 3: Seed Round

Assumption:

- raise: `$2M`
- pre-money valuation: `$12M`
- effective dilution from priced round: about `14.3%`

## Base Case Ownership Snapshot

Illustrative only:

- Founder: about `67% to 71%`
- Pre-seed SAFE investors: about `9% to 11%`
- Seed investors: about `14% to 16%`
- Option pool: about `8% to 10%`

## Conservative Scenario

### Assumptions

- pre-seed: `$500k` at `$5M` cap
- option pool: `10%`
- seed: `$1.5M` at `$10M` pre-money

### Outcome

- founder likely remains around `68% to 72%`

## Aggressive Scenario

### Assumptions

- pre-seed: `$1M` at `$7M` cap
- option pool: `12%`
- seed: `$3M` at `$12M` pre-money

### Outcome

- founder could fall toward `58% to 64%`

This is still viable, but less desirable for a solo founder if the round was raised before strong traction.

## Recommended Cap Table Limits

### Before Seed

Try to avoid:

- more than `15%` sold via SAFEs and notes combined unless pricing is exceptional

### By End Of Seed

Try to keep founder ownership above:

- `60%` preferred
- `55%` minimum acceptable in a strong strategic scenario

## Option Pool Guidance

### Recommended Initial Range

- `8% to 10%`

### Expand Only When Needed

Create enough room for:

- founding engineer hires
- ML talent
- key early design / product support

Do not blindly accept a large pre-money option pool if hiring needs do not support it.

## SAFE Management Rules

- use post-money SAFE instruments for clarity
- track every SAFE in one cap table model
- do not issue multiple uncapped instruments
- understand stacked dilution before signing the next note

## Share Classes To Plan For

- common stock for founder
- option pool for employees / advisors
- SAFE holders converting later
- preferred shares after priced round

## Documents Needed Alongside The Cap Table

- incorporation records
- stock issuance records
- SAFE agreements
- board / founder approvals
- IP assignment agreements
- option plan documents later

## Recommendation

For Provance, the healthiest near-term structure is:

- start simple
- raise a disciplined pre-seed
- preserve meaningful founder ownership
- create a modest option pool
- avoid over-raising before proof exists
