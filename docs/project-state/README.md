# Provance Project State

This directory is the living documentation set for the current state of the Provance product, platform, roadmap, and engineering priorities.

It exists so that substantial analysis, reviews, planning, and project understanding live in the repository rather than only in chat history.

## Purpose

These documents should be updated whenever major changes occur in:

- product scope
- architecture
- implementation status
- roadmap priorities
- security posture
- release readiness
- design direction
- technical debt
- open questions

## Canonical Relationship To Existing Docs

This directory is a living operational summary layer.

It should stay aligned with the deeper source-of-truth documents already in the repository, especially:

- `README.md`
- `docs/engineering/CURRENT_IMPLEMENTATION_STATUS.md`
- `docs/engineering/ENGINEERING_HANDOFF_2026-07-07.md`
- `docs/engineering/PHASE_TASK_LIST.md`
- `docs/engineering/CREDENTIALS_AND_ENVIRONMENT_VARIABLES.md`
- `docs/engineering/DEPLOYMENT_AND_AUTH_STRATEGY.md`
- `docs/changelogs/CHANGELOG.md`

If a conflict appears:

1. update the stale document
2. resolve the conflict explicitly
3. do not let both versions remain ambiguous

## Maintenance Rules

- Update these docs whenever a major engineering, product, design, or process change is made.
- If a release changes product behavior, update the relevant project-state documents before or alongside release closeout.
- If a decision affects long-term direction, record it in `decision-log.md` and, where appropriate, in `docs/decisions/`.
- If a document becomes obsolete, mark it as historical rather than leaving it misleading.

## Current Priority

The immediate priority is:

- landing page redesign and refinement
- brand identity improvement
- typography and font improvements
- improved visual hierarchy
- premium polish
- better responsiveness
- stronger storytelling and messaging

Security hardening, backend cleanup, and RLS expansion remain important, but they are not the active execution priority for the current phase.
