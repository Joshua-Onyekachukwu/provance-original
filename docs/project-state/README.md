# Provance Project State

Last updated: 2026-07-23

This directory is the living current-state summary layer for Provance.

It should help a new engineer understand:

- what exists now
- what is actively being built
- what is deferred
- what risks and blockers remain

## Canonical Relationship To Other Docs

This directory should stay aligned with:

- `README.md`
- `docs/README.md`
- `docs/roadmap/MASTER_DEVELOPMENT_ROADMAP.md`
- `docs/engineering/PHASE_TASK_LIST.md`
- `docs/engineering/CURRENT_IMPLEMENTATION_STATUS.md`
- `docs/architecture/TECHNOLOGY_STACK_REFERENCE.md`

If a conflict appears:

1. update the stale document
2. do not leave both versions active
3. mark historical material clearly when needed
